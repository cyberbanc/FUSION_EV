M9 FUSION EV 1.1.0 — ПОРЯДОК ЗАМЕНЫ

1. Не удаляйте PostgreSQL и не очищайте таблицы.
2. Полностью замените все файлы GitHub содержимым этого пакета.
3. Полностью замените Railway Variables файлом из отдельного Railway-пакета.
4. Выполните Redeploy.
5. После запуска проверьте /health и /payout/calibration.

Приложение автоматически добавляет недостающие столбцы и сохраняет старую историю.
Tilda менять не требуется.
